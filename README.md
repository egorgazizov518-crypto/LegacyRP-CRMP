# Legacy RP - CRMP Mobile RP Server

Полноценный Full RP мод для CRMP Mobile.

## Основные фичи
- Жёсткая реалистичная экономика
- Система группировок с двумя путями развития
- Продвинутая система образования (Колледж, Школа Полиции, Академия ФСБ)
- Документы, ксивы, трудовая книжка
- Legacy Coins (LC) — донат валюта (1 LC = 1000 вирт)
- Большое количество кастомных скинов
- Лаунчер в белом + красном стиле
